## âš™ï¸ How to Use

1. Open the CloudStream app.  
If you don't have it, download it [HERE](https://github.com/recloudstream/cloudstream/releases/tag/pre-release).  
2. Go to **"Settings"** > **"Extensions"**.
3. Tap on **"Add Repository"**.
4. Enter the Hatsune repository URL:
   `https://raw.githubusercontent.com/HatsuneMikuUwU/cloudstream-extensions-uwu/refs/heads/master/repo.json`
5. Tap **"Add"**.
6. The available extensions will appear in the list.  
7. Select the extension you want to install and tap **"Install"** âœ….

For those who want to build it themselves:

2. Install **Gradle** (version 8.12 is recommended).  
3. Install the **Android SDK Manager** (latest version is recommended).
